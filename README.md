# Henrietta’s Cafe – Popmenu Web Designer Exercise

Static responsive landing page built with **HTML + CSS only** (no frameworks, no JS).

## Preview / Run

This is a static project. Open `index.html` directly in the browser or use any local server.

### Option 1: Open file
- Double click `index.html`

### Option 2: VS Code Live Server
- Install “Live Server”
- Right click `index.html` → **Open with Live Server**
